cmsimde_and_stunnel

在 cmsimde 倉儲中以 waitress 啟用動態網站:

waitress_for_wcm2022.py

# 暫時關閉系統 Proxy 設定後, 安裝 waitress 模組
# pip install waitress
from waitress import serve
import sys
import os
# 將 cmsimde 目錄納入模組搜尋
sys.path.append(os.path.join("./cmsimde"))

# 導入 flaskapp
import flaskapp

# run cmsimde dynamic site with production waitress
# 必須配合 https 伺服器承接 port 5000 送出的資料
serve(flaskapp.app, host='127.0.0.1', port=5000, url_scheme='https')

將 cmsimde 動態網站所需的 stunnel.conf 設定加入:

; https 連線設定 for nginx
[https]
; 從 port 443 接受外部 IPv6 網路連線
accept  = :::443
; 資料來源從 http://[::1]:9443 取得
connect = ::1:9443
; https 數位簽章
cert = fullchain.pem
key = privkey.pem
; 伺服器採即時反應
TIMEOUTclose = 0

; https 連線設定 for cmsimde 動態網站
[https]
; 從 port 8443 接受外部 IPv6 網路連線
accept  = :::8443
; 資料來源從 http://127.0.0.1:5000 取得
connect = 5000
; https 數位簽章
cert = fullchain.pem
key = privkey.pem
; 伺服器採即時反應
TIMEOUTclose = 0

nssm 建立系統服務:

以系統管理者啟動 cmd

利用 python.exe 執行 waitress_for_wcm2022.py

nssm install server_name

nssm 編輯系統服務:

以系統管理者啟動 cmd

nssm edit server_name

nssm 刪除系統服務:

以系統管理者啟動 cmd

nssm remove server_name
