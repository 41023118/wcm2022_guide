這個章節在協助學員了解與 Virtualbox 有關的相關安裝與設定內容:

電腦輔助設計室於 2022/05/11 所安裝的 Virtualbox 為 6.1.28 版, 可以從系統的登錄檔案中查核:

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Oracle]

[HKEY_LOCAL_MACHINE\SOFTWARE\Oracle\VirtualBox]
"Version"="6.1.28"
"VersionExt"="6.1.28"
"InstallDir"="C:\\Program Files\\Oracle\\VirtualBox\\"
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\SOFTWARE\Oracle]

[HKEY_CURRENT_USER\SOFTWARE\Oracle\VirtualBox]

[HKEY_CURRENT_USER\SOFTWARE\Oracle\VirtualBox\Install]
@="installed"

當使用者在 Windows 操作系統安裝設定 Virtualbox 虛擬主機後, 以虛擬主機名稱命名 (以 NXServer 為例) 的目錄將會配置在 "C:\Users\%username%\VirtualBox VMs" 目錄中, 其中的 %username% 代表安裝 Virtualbox 所使用的使用者名稱.

而該安裝目錄 NXServer 中, 將配置 NXServer.vbox 與 NXServer.vbox-prev 等兩個檔案. 其中包含 Machine uuid 與 HardDisk uuid 等相關設定資訊.

至於與 Virtualbox 系統設定檔案則位於 C:\Users\%username%\.VirtualBox\VirtualBox.xml

儘管使用者可以透過上述設定檔案, 採手動配置虛擬主機, 將 .vdi 與設定檔案透過可攜隨身碟配置, 但 VirtualBox 每次啟動仍會以啟動後的暫存設定蓋過上述兩個設定檔案.
